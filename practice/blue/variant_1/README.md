Практическая реализация проекта

### Описание проекта

Тебе нужно доработать Telegram-бота для получения информации о странах мира. Бот должен уметь:
- Получать информацию о стране по её названию через внешний API
- Сохранять избранные страны пользователя
- Показывать список сохранённых стран с помощью интерактивных кнопок
- Обрабатывать ошибки при работе с API

### Структура проекта

Проект уже содержит базовую структуру и частично реализованный код. Товя задача — завершить реализацию функций, помеченных как `TODO`, следуя указанным требованиям.

### Технические требования

- Python 3.12+
- Библиотеки: `pyTelegramBotAPI` (telebot), `requests`
- Использовать внешний API для получения информации о странах
- Бот должен быть реализован синхронно

### Что нужно сделать

1. Изучи структуру проекта
2. Найди все `TODO` в коде
3. Реализуй недостающий функционал согласно описанию в комментариях
4. Убедись, что код работает корректно и обрабатывает ошибки
5. Запусти автотесты для проверки реализации 
6. Сгенерируй отчет `TEST_REPORT.md` - это обязательный артефакт для сдачи

### Важные замечания

- Не изменяй существующий код без необходимости
- Следуй структуре проекта
- Обращай внимание на обработку ошибок
- Код должен быть читаемым и структурированным

### Проверка выполнения

После реализации всех TODO запусти автотесты:

```bash
# Установи зависимости
pip install -r requirements.txt

# Запусти тесты и сгенерируйте отчет
./run_tests.sh  # Linux/Mac
# или
run_tests.bat  # Windows
```

Будет создан файл **`TEST_REPORT.md`** с результатами проверки всех образовательных результатов. Этот файл является обязательным артефактом для сдачи контрольной работы.

Подробная инструкция по тестированию находится в файле `TESTING.md`.
