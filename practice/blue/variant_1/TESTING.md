# Инструкция по запуску автотестов

## Описание

Автотесты проверяют выполнение образовательных результатов (ОРов) уровня "Умеет" для практической части контрольной работы.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск тестов

### Вариант 1: Использование скрипта (рекомендуется)

**Linux/Mac:**
```bash
chmod +x run_tests.sh
./run_tests.sh
```

**Windows:**
```cmd
run_tests.bat
```

### Вариант 2: Ручной запуск

```bash
# Запуск всех тестов с генерацией JSON отчета
pytest tests/ --json-report --json-report-file=test_results.json -v

# Генерация отчета
python tests/generate_report.py
```

## Результаты

После выполнения тестов будет создан файл **`TEST_REPORT.md`** с детальным отчетом по каждому образовательному результату.

### Тесты не находят функции
Проверь, что все TODO реализованы и функции экспортированы правильно.

### Ошибки в тестах
Внимательно прочитай сообщения об ошибках в консоли и в файле `TEST_REPORT.md`.
